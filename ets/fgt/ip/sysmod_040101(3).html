HOTTAP "TMS" System Enhancements

FGT is pleased to announce, based on customer feedback, effective April 2, 2001 the following system enhancements will be implemented.

Enhanced Nomination and Scheduling Views

  • Scheduled Quantity View:

The scheduled quantities view will now display information by cycles. To view scheduled quantities by cycle, select the appropriate contract, gas date and cycle on the scheduled quantities open box. The system will default to the latest completed cycle for that gas date. The scheduled quantities will display for the cycle selected. You may also select a new cycle directly from the scheduled quantities screen by clicking on the drop down cycle field that has been added to the scheduled quantities screen. To select a different contract or gas day, return to the open box and select a new contract, gas date, and cycle.

  • View All Pool View:

On pooling contracts, sub totals by pool or zone are now available in the View All Icon on the nomination maintenance screen as well as the schedule quantities screen. The sub totals will default by pool/zone totals with the ability to sort within each pool/zone by using the sort table found under the Tools menu.

  • Nomination & Scheduled Quantity Views:

On the nomination maintenance view and the scheduled quantities view, we have added the "Created by" and "Updated by" fields. These new data fields will identify who created the transaction as well as who updated the transaction last. To access the new data fields, scroll all the way to the right of the nomination screen or scheduled quantities screen.

New Reduction Reason Codes

New defined Reduction Reason Codes will be used during the scheduling process. These new codes are added to the Scheduled Quantity screen, Path Detective screen, Shipper Scheduled Quantity Reports, and Operator Scheduled Quantity Reports.New Reduction Reason Codes

RLI¾ Receipt Location Invalid

DLI¾ Delivery Location Invalid

FPC¾ Fuel Percent Change

CQC¾ Contract MDQ Change

SCT ¾ Service Requester Contract Terminated

RQC¾ Receipt Point MDQ Change

DQC¾ Delivery Point MDQ Change

IQC¾ Max Daily Injection Quantity Change

WQC¾ Max Daily Withdrawal Quantity Change

SBC¾ Storage Balance Change
